# ClawSide 🦞

Supercharge your browsing with local AI — translate, summarize, and ask about any webpage.

## What It Does

ClawSide brings AI directly to your browser via any local LLM/Agent with OpenAI-compatible API (such as OpenClaw, Hermes-Agent, Ollama etc.). No cloud, no fees, just fast local AI.

## Features ✨

- **Translate** 🌍 — Select text → instant translation in any language
- **Summarize** 📖 — Get quick summaries of any page or selection
- **Ask AI** 🧠 — Chat with AI about page content for instant answers
- **Global Translation** 🌐 — Translate entire page with one click
- **Floating Bubble** 💬 — Appears when you select text anywhere
- **Side Panel** 📋 — Full chat with page context for smarter answers

## Why Install 🚀

- **Free forever** 💰 — No subscriptions, no fees
- **Privacy-first** 🔒 — Runs locally via your own LLM
- **Works anywhere** 🌎 — Any webpage, any language
- **Saves time** ⏱️ — No tab-switching or copy-pasting
- **Flexible** 🔌 — Works with any OpenAI-compatible LLM

## Get Started 🛠️

1. Add ClawSide to Chrome
2. Run any OpenAI-compatible LLM locally
3. Select text anywhere → translate, summarize, or ask!
