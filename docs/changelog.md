# Changelog

All notable changes to ClawSide will be documented in this file.

## [1.0.0] - 2026-04-10

### Added
- Initial release
- Floating bubble for text selection (translate, summarize, ask)
- Side panel with full functionality
- Page summarization with context
- Chat interface for Q&A about pages
- History tracking
- Settings management (language, appearance, prompts)
- Multi-language support (i18n)
- Debug info in About tab

### Features
- Chrome side panel extension bridging browser to local OpenClaw Gateway
- Translate selected text to multiple languages
- Summarize current page content
- Ask questions about page context
- Global page translation via radial menu
- Per-tab+URL conversation history
